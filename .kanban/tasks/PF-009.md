## Technical Notes

```python
# Target: minimal script, run from terminal
# File: ai-service/scripts/hello_llm.py
import anthropic

client = anthropic.Anthropic()  # reads ANTHROPIC_API_KEY from env
message = client.messages.create(
    model="claude-sonnet-4-20250514",
    max_tokens=1024,
    system="You are a financial data assistant.",
    messages=[{"role": "user", "content": "Summarize this text: ..."}]
)
```

- Start with `claude-sonnet-4-20250514` for cost efficiency during learning
- Experiment with temperature (0.0 for deterministic, 0.7 for creative)
- Log token usage from response metadata
