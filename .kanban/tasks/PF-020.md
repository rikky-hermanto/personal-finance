## Objective

Expose the RAG pipeline as a user-facing API endpoint. The frontend query bar uses this for natural language financial queries.

## Acceptance Criteria

- [ ] `POST /api/search` endpoint with natural language query
- [ ] Returns: answer text, source transactions, confidence
- [ ] Frontend query component integrated
- [ ] Response streaming for long answers (SSE or chunked)
- [ ] Query history stored for analytics
- [ ] Rate limiting on AI-powered endpoints
