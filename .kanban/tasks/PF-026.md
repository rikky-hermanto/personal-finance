## Objective

Add authentication and authorization using Auth0. Secure all API endpoints. Rate limit AI-powered endpoints to prevent abuse.

## Acceptance Criteria

- [ ] Auth0 tenant configured (OAuth 2.0 / OIDC)
- [ ] JWT validation middleware in .NET API
- [ ] Protected endpoints require valid token
- [ ] User context available in all request handlers
- [ ] Rate limiting on AI endpoints (per-user quotas)
- [ ] API key management for service-to-service calls
- [ ] Input sanitization on all AI-facing endpoints
- [ ] Frontend login/logout flow

## Dependencies

- Core features stable (Sprint 1-3 complete)

## Learning Objectives

- OAuth 2.0 / OIDC with Auth0
- JWT validation in ASP.NET Core
- API security best practices for AI applications
- Rate limiting patterns
