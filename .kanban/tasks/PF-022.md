## Objective

Implement an autonomous agent loop using Semantic Kernel on the .NET side. Agent receives a goal, plans steps, executes tools, and synthesizes a response.

## Acceptance Criteria

- [ ] Semantic Kernel integrated into .NET API
- [ ] Agent plugins registered for each API capability
- [ ] Agent loop: receive goal → plan → execute → observe → repeat/answer
- [ ] Execution trace logged for debugging
- [ ] Safety: max iterations limit, cost budget per request
- [ ] Example: "Categorize all uncategorized transactions from December"
