## Objective

Enable multi-turn conversations where the agent maintains context across messages. "Show my December expenses" → "Now break that down by category" → "Which category increased the most vs November?"

## Acceptance Criteria

- [ ] Conversation session management (session ID, message history)
- [ ] Agent maintains context across turns
- [ ] Conversation memory: short-term (current session) + references to data
- [ ] Frontend chat UI component
- [ ] Session cleanup / expiration policy
- [ ] Conversation export / summary
