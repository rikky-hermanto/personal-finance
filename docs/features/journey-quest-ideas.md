# JOURNEY Quest Ideas — Financial Pyramid Alignment

> Brainstormed 2026-05-19. Reference: [5 Piramida Keuangan — Makmur.id](https://www.makmur.id/id/blog/artikel/mengenal-5-piramida-keuangan-dalam-mencapai-tujuan-keuangan-jangka-panjang)
>
> Each level maps to an existing JOURNEY tier. Quests are grouped by difficulty and tagged with the indicator they target.

---

## Level Mapping

| Pyramid Level | JOURNEY Tier | Theme |
|---|---|---|
| L1: Cash Flow + Emergency Fund + Debt | **Foundations** | Survival & control |
| L2: Risk Management & Insurance | **Defense** | Protection |
| L3: Medium-Term Goals | **Growth** | Building |
| L4: Retirement & Passive Income | **Freedom** | Independence |
| L5: Wealth Distribution | **Legacy** | Impact beyond self |

## Quest Design Principles

1. **Show only L-current and L-current+1 quests** — don't overwhelm users with L4 quests when they're on L1.
2. **Always have 1 easy quest active** — users need a quick win to stay engaged, especially after onboarding.
3. **Progress quests > one-shot quests** — streaks and milestones create return visits; one-shot quests feel like checkbox tasks.
4. **Tie every quest to a navigation deeplink** — "Start" should take users directly to the relevant module.
5. **Celebrate graduation explicitly** — when a user completes all L1 quests, the Living Garden should animate meaningfully.
6. **Indonesian context matters** — use IDR milestones (100jt, 500jt, 1M, 5M), BPJS, SBN/ORI references.
